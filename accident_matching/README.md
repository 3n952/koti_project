# 사고 맵매칭 결과

도로 위에서 발생한 사고 지점을 실제 지도 위에 맵매칭하여 시각화한 결과입니다.  
사고가 일어났을 것으로 판단하는 링크를 top5를 산출하고 이에 대해 recall@5를 평가지표로 합니다.  
다음 결과 이미지 예시는 top10에 대한 예시 입니다.  

- 빨간점은 임의의 사고 지점입니다.  
- 빨간선은 정답 도로, 파란선은 top10 도로, 검은선은 top10에서 제외된 도로를 의미합니다.
